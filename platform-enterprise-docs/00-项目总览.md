# Platform Enterprise - 企业级微服务平台总览

> 基于 Spring Boot 4.0 + Vue 3.5 的现代化企业级应用平台

## 1. 项目概述

### 1.1 项目背景

Platform Enterprise 是一套面向中大型企业的数字化转型解决方案，采用最新的 Spring Boot 4.0 和 Vue 3.5 技术栈，结合云原生架构思想，提供高性能、高可用、易扩展的企业级应用开发平台。

### 1.2 设计目标

| 目标 | 描述 |
|------|------|
| **高性能** | 利用 Java 21 虚拟线程，单实例支撑 10000+ 并发连接 |
| **高可用** | 99.99% 系统可用性，支持多活部署 |
| **易扩展** | 模块化设计，支持水平扩展和功能插拔 |
| **安全性** | 零信任安全架构，全链路加密 |
| **可观测** | 完整的日志、指标、链路追踪体系 |

## 4. 核心特性

### 4.1 Spring Boot 4.0 新特性应用

- **虚拟线程**: 默认启用 Java 21 虚拟线程，显著提升 I/O 密集型场景性能
- **模块化架构**: 采用 Spring Boot 4.0 新的模块化 JAR，减少启动时间
- **GraalVM 原生镜像**: 支持编译为原生可执行文件，毫秒级启动
- **JSpecify 空安全**: 全面采用 JSpecify 注解，增强代码健壮性
- **声明式 HTTP 客户端**: 使用新的 `@HttpExchange` 接口简化服务间调用

### 4.2 Vue 3.5 新特性应用

- **Composition API**: 全面采用组合式 API，提升代码复用性
- **TypeScript 5.x**: 完整的类型覆盖，IDE 智能提示
- **Pinia 3**: 轻量级状态管理，支持 SSR
- **Vite 6**: 极速开发体验，秒级热更新
- **useTemplateRef**: 增强的模板引用管理

## 6. 快速开始

### 6.1 环境要求

```yaml
# 开发环境最低要求
JDK: 21+
Node.js: 22+
Docker: 27+
Maven: 3.9+
```

### 6.2 一键启动开发环境

```bash
# 克隆项目
git clone https://github.com/company/platform-enterprise.git

# 启动基础设施
cd platform-docker
docker compose up -d

# 启动后端服务
cd ../platform-backend
./mvnw spring-boot:run -pl platform-gateway

# 启动前端
cd ../platform-ui
pnpm install && pnpm dev
```
